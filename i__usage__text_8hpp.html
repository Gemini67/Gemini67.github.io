Macro to easily define the output function.


Actually creates a class called <c>_class which contains the required print() method, as well a static, global variable called c (of type <c>_class) which can then be passed to evalArguments().

Parameters

pThe position of the text in the usage (see enum UsagePos in class ArgumentHandler)
tThe text to print.
