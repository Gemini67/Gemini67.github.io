Detailed Description

template<class T, template< typename > class C = StaticSingletonCreator>
class celma::common::detail::Singleton< T, C >

Template for creating singleton classes.

The template prevents creating of multiple instances, copying etc.
Usage:

  • Derive the class from this template, with the class itself as template parameter.
  • The class must have a constructor that is protected.
  • The template Singleton must be defined as friend.
