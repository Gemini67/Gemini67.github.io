Detailed Description

template<typename T, typename C>
class celma::common::RangeDest< T, C >

Helper class to store value(s) in a container.

Partial template specialisation is used to provide the same interface for all standard containers.
Default type for all containers that support push_back().

Template Parameters
TThe type of the values stored in the container.
