Detailed Description

template<typename T, T iterEndValue = -1>
class celma::common::RangeGenerator< T, iterEndValue >

Generates all numbers in a range.


The class can also be used to handle 'ranges' with only a single value.
The object can be used just once to create the values in the range. If the same values of the same range are needed multiple times, multiple generator objects must be created.

Template Parameters
TThe data type of the values to be produced by the generator.
iterEndValueIterator end value, must be a value that is outside of the range.
