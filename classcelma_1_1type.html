Detailed Description

template<typename T>
class celma::type< T >

Base template for getting the name of a type.

Partial template specialisation is used to provide the name of a type.
The design with the static method was chosen because the solution with a static member variable led to 'multiple instance' error messages. Maybe this would not happen with C++11x 'constexpr'.
The template was added to the 'std' namespace on purpose.
