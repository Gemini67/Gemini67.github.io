Detailed Description

Helper class to convert an argument string (e.g.

'-n PROC1 -v -d /tmp') into an array of string pointers with the separate arguments.
Afterwards, the string array can be passed to a function that expects parameters in the same format as a main() function.
This function does similar processing regarding multi-word strings: Quotes are removed, but the text within the quotes is passed as one value.

celma::common::ArgString2Array::ArgString2Array ( const std::string &  argstring,
const char *  progname 
)

Constructor, splits the string into separate arguments.


The file name is automatically set in the first string in the array, even if the argument string does not contain the program name.
So, if you want to create an argument string that contains only a free argument (not preceeded by an argument with one or two dashes), you need to set a program name before it.

Parameters
[in]argstringThe argument string to split.
[in]prognameOptional program name to set. If not set, 'programname' is set as mpArgv[ 0].
