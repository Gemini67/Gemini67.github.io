Detailed Description

template<typename T1, typename T2>
class celma::common::MultiSetter2< T1, T2 >

Template to use together with the class ArgumentHandler:
When assign is called, sets the passed value on the first variable, and sets the value specified in the constructor on the second variable.


Use the macro DEST_VAR() to easily initialise objects of this class.
