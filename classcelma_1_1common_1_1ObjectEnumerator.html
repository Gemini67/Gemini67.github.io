Detailed Description

template<typename T, typename C = uint64_t>
class celma::common::ObjectEnumerator< T, C >

Helper class to enumerate all objects of a class.


Use this template with the CRTP: class Counter: public ObjectEnumerator< Counter> This (the CRTP) is necessary to make sure that multiple enumerated classes in the same application are counted separately.

Template Parameters
TThe class to enumerate the objects of.
CThe type to use for the counter.
